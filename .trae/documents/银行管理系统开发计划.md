# 银行管理系统开发计划

## 一、项目概述
开发一个基于Java和MySQL的银行管理系统，包含登录、注册、存款、取款、转账、查看余额和注销账户等功能。

## 二、技术栈
- **开发语言**：Java
- **数据库**：MySQL
- **数据库连接**：JDBC
- **界面框架**：Java Swing

## 三、数据库设计

### 1. 数据库表结构
```sql
CREATE TABLE bank_user (
    account VARCHAR(11) PRIMARY KEY,  -- 账号，11位数字
    name VARCHAR(50) NOT NULL,        -- 姓名
    password VARCHAR(255) NOT NULL,   -- 密码
    phone VARCHAR(11) NOT NULL,       -- 手机号
    id_card VARCHAR(18) NOT NULL,     -- 身份证号
    balance DECIMAL(10,2) DEFAULT 0.00 -- 余额
);
```

## 五、核心功能实现步骤

### 1. 数据库连接工具类（DBUtil）
- 加载数据库驱动
- 建立数据库连接
- 关闭数据库资源

### 2. 用户实体类（User）
- 封装用户信息：账号、姓名、密码、手机号、身份证号、余额

### 3. 数据访问层（DAO）
- UserDao接口：定义用户相关的数据库操作方法
- UserDaoImpl实现类：使用JDBC实现具体的数据库操作

### 4. 业务逻辑层（Service）
- UserService接口：定义用户相关的业务逻辑方法
- UserServiceImpl实现类：实现具体的业务逻辑

### 5. 界面层（View）

#### 登录界面（LoginFrame）
- 账号输入框
- 密码输入框
- 登录按钮
- 注册按钮
- 退出按钮

#### 注册界面（RegisterFrame）
- 姓名输入框
- 密码输入框
- 手机号输入框
- 身份证号输入框
- 随机生成账号按钮
- 注册按钮
- 取消按钮

#### 主功能界面（MainFrame）
- 欢迎信息
- 存款功能
- 取款功能
- 查看余额功能
- 转账功能
- 注销账户功能
- 退出功能

### 6. 主程序入口（App）
- 启动登录界面

## 六、关键功能实现要点

### 1. 随机生成11位账号
- 使用Random类生成11位数字
- 验证账号唯一性

### 2. 密码处理
- 使用简单的MD5加密（可选，课程设计级可直接存储明文）

### 3. 转账功能
- 验证对方账号是否存在
- 验证转账金额是否小于等于余额
- 使用事务处理确保转账的原子性

### 4. 界面设计
- 简洁明了的布局
- 友好的用户提示
- 输入验证
